# 研究室配属シミュレーター

匿名IDと編集キーで参加する、学生同士の研究室配属シミュレーション試作です。

## 起動

```powershell
$env:ADMIN_PASSWORD="任意の管理者パスワード"
& "C:\Users\yushu\.cache\codex-runtimes\codex-primary-runtime\dependencies\node\bin\node.exe" server.js
```

ブラウザで `http://localhost:3000` を開きます。

`ADMIN_PASSWORD` を設定しない場合、管理者パスワードは `admin-reset` です。

## 方針

- 実名・学籍番号は保存しません。
- GPAは数値で入力し、計算にのみ使います。
- 公開結果ではGPAを表示しません。
- 結果はモデルごと、研究室ごとの匿名ID一覧で表示します。
- 編集・削除には匿名IDと編集キーの両方が必要です。
- 管理者画面では個別のGPAや希望順位を表示しません。
- 管理者は受付切替、研究室設定、全参加データ破棄だけを行えます。

## シミュレーションモデル

- GPA順・最大定員モデル
- 最低人数考慮モデル
- 希望順位最適化モデル

希望順位最適化モデルは参加者12人までは全探索し、それ以上では近似として最低人数考慮モデルを表示します。
